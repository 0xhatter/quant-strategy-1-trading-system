## üî¨ Methodology

### Monte Carlo Simulation Details

**Process:**
1. Collected 720 hours of real Hyperliquid data per token
2. Calculated actual return distribution (mean, std, etc.)
3. Bootstrap resampling: randomly sampled returns with replacement
4. Ran 1000 simulated paths per token
5. Calculated metrics for each path
6. Aggregated statistics (mean, median, CI, percentiles)

**What This Measures:**
- Robustness to order of returns (path independence)
- Statistical distribution of outcomes
- Confidence intervals and risk metrics
- Probability of various outcomes

**What This Does NOT Account For:**
- Regime changes (market conditions shifting)
- Black swan events (extreme outliers)
- Model degradation over time
- Transaction costs and slippage
